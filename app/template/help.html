Getting Around the Ship

There are interactive maps near all of the elevators. However, interactive maps are slow, so if you can learn some basics of ship navigation, you will be much faster.

  • Facing Fore, Port is Left. Starboard is Right. Port has the same number of letters as Left. Right and Starboard have more letters than Left and Port.
  • Look at the hundreds place of room numbers. If it’s 4 or lower, you are in a Starboard corridor. If it’s 5 or higher, you are in a Port corridor.
  • In some cases, you might be in an aft-most area, or an offshoot hallway. You’ll probably know if you’re in one of these places or not.
  • Room numbers get larger the farther Aft you are, smaller the farther Fore you are.
  • Even number rooms are exterior. Odd number rooms are interior.
  • Pay attention to the art in the stairwells, and note which are fore and which are aft.
  • Maps in the hallways are always oriented to the direction of the ship.
Events in CruiseMonkey

There are 3 kinds of events:

  • Official Events: Events that are an official part of the JoCo Cruise Crazy schedule.
  • Unofficial Events: "Shadow cruise" events that are created by other sea monkeys.
  • My Events: Events that you have either created yourself, or that you have favorited in the Official or Unofficial event lists.


Note: you must log using your Twit-Arr account in to be able to create, edit, or favorite events.
